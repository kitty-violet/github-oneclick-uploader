# Changelog

## 0.1.1 - 2026-06-06

- Fixed the project setup layout so the Safety Check, GitHub Login, and Upload to GitHub buttons are visible.
- Made the primary next-step prompt clearer after selecting a project folder.
- Renamed the action buttons to make the upload flow easier to follow.
- Added an empty safety-result row when no findings are detected.

## 0.1.0 - 2026-06-05

- Added WinForms uploader UI.
- Reworked the default experience into a polished desktop dashboard.
- Added a custom app icon and embedded it in the executable.
- Added portability documentation for open-source distribution.
- Added command-line `login`, `check`, and `upload` commands.
- Improved GitHub device login guidance in the GUI.
- Added Git and GitHub CLI checks.
- Added secret and local-data scanning.
- Added GitHub repository creation and push flow.
- Added open-source project docs, scripts, and CI.
